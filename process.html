Arbetsprocess

  1. Jag började med att öppna terminalen och skapa en ny projektmapp. Detta genom att skriva in nmp create vite@latest Laboration-1-Vite-Automatisering. Därefter fyllde jag i det formulär som dyker upp.
  2. Installerade sedan SASS till projektet. Detta gjorde jag genom att i terminalen skriva npm install sass --save-dev.
  3. Skapade ett repository på Github. Kopplade sedan mitt projekt till Github och gjorde min första commit. Detta gjorde jag genom git init, git add, git push och git commit -m "".
  4. Gjorde sedan en ny branch i mitt repository, en brach som döptes till development. Detta gjorde jag genom att i terminalen skriva git checkout -b development / git push -u origin development.
  5. Skapade webbplatsen med hjälp av HTML, CSS och JavaScript.
